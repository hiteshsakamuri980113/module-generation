# React + Vite + TypeScript + Tailwind

Created by scaffold script. Quick start (PowerShell):

```powershell
cd "C:\Users\MY PC\Downloads\module-generation\react-vite-tailwind"
npm install
npm run dev
```

- `npm run dev` — starts the Vite dev server (opens in browser)
- `npm run build` — build for production
- `npm run preview` — locally preview the production build

Notes:
- Tailwind is configured in `tailwind.config.cjs` and `postcss.config.cjs`.
- Main source is in `src/`.
